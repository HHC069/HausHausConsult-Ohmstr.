# GitHub Pages Deployment

## Dateien hochladen

Den kompletten Inhalt dieses Ordners in die Root des Repositories hochladen, nicht den Ordner selbst.

Erwartete Root-Struktur:

```text
index.html
index.inline.html
assets/
.nojekyll
robots.txt
404.html
README.md
DEPLOY_GITHUB_PAGES.md
landingpage_quality_check.json
```

## GitHub Pages aktivieren

1. Repository öffnen
2. Settings -> Pages
3. Source: Deploy from a branch
4. Branch: main
5. Folder: /root
6. Save
